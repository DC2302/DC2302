## Virality pre-flight

The Higgsfield `virality_predictor` tool is capped at ≤16s inputs and this is a 40s asset, so it
was skipped per instruction — this is a manual hook/retention read instead.

- **Hook (0-2s):** Strong. Big D breaks the fourth wall and physically waves the rig in —
  "Rollin' through the Permian? Pull in here." This is the brief's own "strongest" pick
  (pattern-interrupt via motion + direct address, not fear/tension) and it's a good one: it's
  concrete, on-brand, and gives the viewer an instruction, not just a mood. No re-cut needed here.
- **Retention risk — real but structural, not a hook problem:** 40s (4×10s blocks) is on the long
  side for TikTok/Reels/Shorts, where top performers usually front-load payoff inside 15-20s. The
  brotherhood beat (block 2, 10-20s) is the softest section — it's emotional/values messaging with
  no new visual proof yet, which is the most likely drop-off point. Block 3 (shade/grass/grills,
  20-30s) reintroduces visual novelty and should pull attention back before the CTA.
- **CTA/close (30-40s):** Clean — verbal CTA, bilingual sign-off, and the logo end-card land
  together. No issue.
- **Net:** Ship as-is. This is a lead-gen piece that needs the full arc (welcome → brotherhood →
  proof → CTA) to earn the click, so the length is a defensible tradeoff, not a defect. If
  platform analytics later show a hard drop at ~10-15s, the fix is a tighter re-edit of block 2,
  not a new hook — flagging for future iteration, not blocking this post.

## Posting plan

1. **TikTok first**, same day — fastest discovery loop for a cold hook like this, and the
   platform most forgiving of a direct-address, lower-polish mascot style.
2. **Instagram Reels ~2-4 hours later** — reuse the same cut; stagger so it doesn't look like a
   single cross-post blast, and so any early TikTok comments/reactions can inform whether the
   caption needs a quick tweak before the IG post.
3. **YouTube Shorts same day, evening** — Shorts distribution rewards freshness less urgently than
   TikTok/Reels, so it can trail by most of a day without losing much.
4. **Facebook next day** — post to the CADD Page first, then share into 2-3 relevant Permian
   Basin/owner-operator trucking groups (respecting each group's self-promo rules — lead with the
   value/welcome, not just a link drop). Facebook audiences here skew toward longer dwell/read
   time, so a one-day lag to let the short-form platforms seed initial engagement/proof-of-concept
   is fine.
